Deacon

A Rust implementation of the Development Containers CLI.

Quick Install

curl -fsSL https://get2knowio.github.io/deacon/install.sh | bash

Install Options

# Install the latest pre-release
curl -fsSL https://get2knowio.github.io/deacon/install.sh | bash -s -- --prerelease

# Install a specific version
curl -fsSL https://get2knowio.github.io/deacon/install.sh | DEACON_VERSION=0.2.0-rc.2 bash

# Install to a custom directory
curl -fsSL https://get2knowio.github.io/deacon/install.sh | DEACON_INSTALL_DIR=~/.local/bin bash

# Force overwrite without prompt
curl -fsSL https://get2knowio.github.io/deacon/install.sh | DEACON_FORCE=true bash
